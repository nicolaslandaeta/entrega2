# Alke Wallet

Alke Wallet es una aplicación web que permite a los usuarios gestionar sus finanzas personales, incluyendo funcionalidades como depósito de fondos, envío de dinero y visualización de transacciones.

## Funcionalidades

- **Inicio de Sesión**: Los usuarios pueden iniciar sesión con su nombre de usuario y contraseña.
- **Depósito de Fondos**: Permite a los usuarios depositar dinero en su cuenta.
- **Envío de Dinero**: Los usuarios pueden enviar dinero a otros contactos.
- **Últimos Movimientos**: Visualización de las transacciones recientes.

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript (jQuery)
- Bootstrap
- Local Storage (para persistencia de datos)

## Instalación y Configuración

1. Clona el repositorio:
    ```bash
    git clone https://github.com/tu-usuario/alke-wallet.git
    cd alke-wallet
    ```

2. Abre el proyecto en VSCode:
    ```bash
    code .
    ```

3. Inicia Live Server para visualizar el proyecto:
    - Instala la extensión Live Server en VSCode si no la tienes.
    - Haz clic en "Go Live" en la esquina inferior derecha de VSCode.

## Uso

1. **Inicio de Sesión**:
    - Abre `index.html` en tu navegador.
    - Ingresa las credenciales de usuario (ejemplo: usuario: `mimolas`, contraseña: `123momiaes`).

2. **Depósito de Fondos**:
    - Navega a la página de depósito.
    - Ingresa el monto a depositar y haz clic en "Realizar depósito".

3. **Envío de Dinero**:
    - Navega a la página de envío de dinero.
    - Agrega un nuevo contacto y selecciona el contacto al que deseas enviar dinero.
    - Ingresa el monto y haz clic en "Enviar dinero".

4. **Últimos Movimientos**:
    - Navega a la página de últimos movimientos para ver las transacciones recientes.
